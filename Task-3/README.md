## Project Overview
The Temperature Converter application is designed to provide a simple and interactive way to convert temperatures.  
It includes:

- Input field for entering the temperature
- Dropdown menus to select the source and target units (Celsius, Fahrenheit, Kelvin)
- A "Convert" button to calculate the result
- Display area to show the converted temperature

The application ensures **accuracy** and **real-time conversion** with user-friendly interface.

---

## Features
- **Convert Between Units** – Supports Celsius, Fahrenheit, and Kelvin
- **Responsive Design** – Works well on desktops, tablets, and mobile devices
- **Real-time Conversion** – Shows results immediately after clicking "Convert"
- **User-friendly Interface** – Simple layout with clear instructions
