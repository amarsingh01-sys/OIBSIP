## Project Overview
This portfolio is a clean and modern website that includes:

- Hero section introducing the developer
- About Me section
- Skills section with animated skill bars
- Projects section with images and hover effects
- Experience & Education timeline
- Contact section

The design is fully **responsive** and uses **interactive hover effects** for a modern look.

---

## Features
- **Responsive Layout** – Works on desktop, tablet, and mobile devices
- **Interactive Effects** – Hover animations on projects and buttons
- **Modern Design** – Clean typography, spacing, and color palette
- **Easy to Customize** – Update content, images, skills, and colors easily

---

## Technologies Used
- **HTML5** – Website structure
- **CSS3** – Styling and layout (Flexbox & Grid)
- **CSS Animations & Transitions** – Skill bars and hover effects
- **Responsive Design** – Mobile-first approach
